图片上传配置演示

1. 轮播图配置示例

在 image-config.js 中配置：


carousel: [
    {
        src: "images/carousel/slide1.jpg",
        title: "自然风光",
        description: "捕捉大自然的美丽瞬间"
    },
    {
        src: "images/carousel/slide2.jpg",
        title: "城市建筑", 
        description: "记录城市的独特魅力"
    }
]

2. 图集配置示例

在 image-config.js 中配置：


nature: {
    title: "自然风光",
    cover: "images/nature/cover.jpg",
    images: [
        "images/nature/1.jpg",
        "images/nature/2.jpg",
        "images/nature/3.jpg"
    ]
}

3. 文件目录结构


images/
├── carousel/
│   ├── slide1.jpg
│   ├── slide2.jpg
│   └── slide3.jpg
├── nature/
│   ├── cover.jpg
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
└── city/
    ├── cover.jpg
    ├── 1.jpg
    └── 2.jpg

4. 图片规格建议

  • 轮播图: 1200×600 像素，JPG格式
  • 封面图: 400×300 像素，JPG格式
  • 内容图: 800×600 像素，JPG格式
  • 文件大小: 单张不超过500KB
